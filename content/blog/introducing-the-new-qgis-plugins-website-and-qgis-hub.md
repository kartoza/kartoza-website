## Revamped QGIS Plugins Website

The [QGIS Plugins website](https://plugins.qgis.org/) features a modernized interface with several key improvements:

1. Fresh UI aligned with latest QGIS branding
2. Enhanced browsing with improved plugin categorization
3. Detailed plugin pages showing ratings, download counts, and descriptions
4. Improved search and filtering options
5. More intuitive submission process for developers

The redesigned site offers both grid and table view options for browsing plugins, along with enhanced search and details functionality.

## Introducing QGIS Hub

A new platform launched at [https://hub.qgis.org](https://hub.qgis.org) serves as a dedicated space for sharing QGIS resources including styles, 3D models, geopackages, project files, and QLR files.

**Key features include:**

1. Visually appealing homepage with featured resources
2. Well-organized list view for browsing assets
3. Detailed resource pages with previews and descriptions
4. Advanced search functionality
5. Seamless submission process for contributors
