## Overview

This comprehensive course teaches participants how to design and develop professional web mapping applications using open source JavaScript libraries. Participants will learn fundamental web development technologies (HTML, CSS, JavaScript) in the context of mapping applications, explore popular mapping libraries including Leaflet and OpenLayers, and apply cartographic principles to create effective and beautiful web maps. The course covers consuming web map services, adding interactivity, creating custom controls, implementing responsive design, and optimizing performance. Through hands-on projects, participants will build fully functional web mapping applications that can be deployed in production environments.

## What You Will Learn

- Web development fundamentals (HTML, CSS, JavaScript)
- Introduction to Leaflet and OpenLayers libraries
- Consuming OGC web services (WMS, WFS, WCS)
- Working with tile services and basemaps
- Adding markers, popups, and custom overlays
- Implementing user interactions and event handling
- Creating custom map controls and widgets
- Styling vector data with dynamic symbology
- Implementing search and geocoding functionality
- Building responsive maps for mobile devices
- Cartographic principles for web mapping
- Performance optimization techniques
- Integrating spatial data from APIs and databases
- Deploying web mapping applications

## Who Should Attend

This course is ideal for web developers entering the geospatial field, GIS professionals wanting to publish their data on the web, cartographers transitioning to digital media, and anyone interested in creating interactive web maps. Participants should have basic computer skills and be comfortable learning technical concepts. While prior programming experience is helpful, the course starts with fundamentals and builds progressively. The course is suitable for designers, developers, GIS analysts, and communications professionals.

## Duration

3 days (on-site or remote)
