## Overview

Ogun State implemented their inaugural digital Spatial Data Infrastructure (SDI) beginning in 2012. Kartoza was the core developer of the Land Information and Land Administration Systems at its centre. The infrastructure stores data in PostGIS and publishes via GeoServer and PHP using a customised 1map platform implementation.

![Ogun State LIS](/img/portfolio/ogun-state-lis.png)

## Deliverables

- Requirements analysis and system design
- Localised 1Map instance implementation
- Geo-portal dataset preparation and publication
- QGIS cadastral plugin development for beacon and parcel capture
- Staff training programmes
- COGO (bearing and distance) capture capabilities

## Technology

- PostGIS
- GeoServer
- PHP
- QGIS Python plugin
- PostgreSQL
- 1Map platform
