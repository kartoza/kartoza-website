## Overview

Kartoza's Software Development services deliver custom geospatial solutions tailored to your specific needs. We build robust, scalable applications using modern technologies and open source platforms.

From simple QGIS plugins to complex enterprise web applications, our experienced development team brings your GIS vision to life.

## What We Build

- Custom web mapping applications
- Mobile GIS apps (iOS, Android, cross-platform)
- QGIS plugins and extensions
- PostGIS database applications
- Spatial data pipelines and ETL
- API development and integrations
- Dashboard and visualization tools
- Field data collection solutions

## Technologies We Use

- **Frontend**: React, Vue.js, OpenLayers, Leaflet, MapLibre
- **Backend**: Python, Django, Flask, Node.js
- **Desktop**: PyQGIS, Qt, C++
- **Mobile**: React Native, Flutter
- **Database**: PostgreSQL/PostGIS, SQLite
- **DevOps**: Docker, Kubernetes, CI/CD
