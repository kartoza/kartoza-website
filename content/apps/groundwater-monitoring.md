## Overview

The Groundwater Monitoring app enables users to georeference and register groundwater monitoring stations and submit data to the Global Groundwater Monitoring Network (GGMN) Portal managed by UN-IGRAC. The app streamlines field data collection for groundwater monitoring programmes worldwide.

![Groundwater Monitoring](/img/apps/groundwater.png)

## Features

- Georeference monitoring stations
- Register new groundwater stations
- Submit data to the GGMN Portal
- Field data collection workflows
