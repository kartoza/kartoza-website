## Overview

The SAGTA Map Exporter is a specialized QGIS plugin that enables land surveyors in South Africa to produce compliant cadastral and surveying maps according to SAGTA (South African Geomatics Technicians Association) standards.

This tool streamlines the map production process, ensuring that outputs meet regulatory requirements while reducing manual effort and errors.

## Key Features

- Automated layout generation
- Compliant symbology and styling
- Standard title block templates
- Coordinate grid generation
- Scale bar and north arrow placement
- Multi-sheet map support
- PDF and print output
- Batch processing capabilities

## Compliance Standards

- SAGTA map specifications
- Chief Surveyor-General requirements
- Cadastral diagram standards
- General plan requirements
- Sectional title plan formats

## Benefits for Surveyors

### Time Savings
Automate repetitive layout tasks and focus on survey accuracy.

### Consistency
Ensure every map meets standards with templated outputs.

### Quality
Reduce errors with automated checks and validation.

### Flexibility
Work within the powerful QGIS environment with full GIS capabilities.

## Requirements

- QGIS 3.x (Long Term Release recommended)
- Valid survey data in supported formats
- Appropriate coordinate reference systems
