## Overview

The SAGTA Map Exporter is a specialized QGIS plugin developed by Kartoza that enables land surveyors and GIS professionals to produce cadastral and surveying maps that comply with South African standards and regulations.

Designed in collaboration with practising land surveyors and the surveying community, this plugin streamlines the map production workflow, ensuring that outputs meet the requirements of the Surveyor-General and other regulatory bodies.

## Key Features

### Standards Compliance

- **Surveyor-General Requirements**: Maps formatted according to official specifications
- **SAGTA Guidelines**: Compliance with South African Geomatics Institute standards
- **Deed Office Formats**: Output formats accepted by the Deeds Office
- **Coordinate Systems**: Support for South African coordinate reference systems (Lo systems, WGS84, Hartebeesthoek94)
- **Scale Requirements**: Automatic scaling to standard cadastral map scales

### Map Templates

- **General Plan Templates**: Standard templates for general plans
- **Sectional Plan Templates**: Templates for sectional title developments
- **Diagram Templates**: Survey diagram layouts
- **Custom Templates**: Create and save your own templates
- **Template Library**: Pre-configured templates for common use cases

### Automated Layout

- **Title Block Generation**: Automatic population of title blocks with survey data
- **North Arrow Placement**: Correctly positioned and styled north arrows
- **Scale Bar Formatting**: Appropriate scale bars for cadastral maps
- **Legend Generation**: Automatic legend creation from layer symbology
- **Coordinate Grid**: Grid lines and coordinate labels per specifications

### Data Integration

- **Parcel Data Import**: Import cadastral parcel data from various sources
- **Beacon Information**: Manage and display beacon coordinates
- **Survey Data**: Integration with survey calculation outputs
- **Attribute Tables**: Include required attribute information
- **Metadata Embedding**: Embed required metadata in outputs

### Export Options

- **PDF Export**: High-quality PDF output for printing and submission
- **DXF/DWG**: CAD format export for interoperability
- **Image Formats**: PNG, TIFF, and other image formats
- **Print Layouts**: Direct printing with correct margins and sizing
- **Batch Export**: Export multiple maps in a single operation

## Workflow

### 1. Prepare Your Data

Load your cadastral and survey data into QGIS:
- Parcel boundaries
- Beacon points
- Servitudes and restrictions
- Reference cadastral data
- Topographic features

### 2. Configure the Export

Open the SAGTA Exporter and configure:
- Select the appropriate template
- Define the map extent
- Set the output scale
- Enter title block information
- Configure coordinate display options

### 3. Generate and Review

The plugin generates a print layout with:
- All required elements correctly positioned
- Compliant symbology and annotation
- Proper coordinate system and projection
- Formatted according to selected template

### 4. Export and Submit

Export your map in the required format:
- Generate PDF for digital submission
- Print hard copies as required
- Export CAD formats for other systems

- **Custom CRS**: Support for other coordinate systems as needed

- Landscape and portrait orientations

## Frequently Asked Questions

**Is this an official SAGTA product?**

No, SAGTA Exporter is developed by Kartoza. The name refers to the standards and guidelines from the South African Geomatics Institute that the plugin helps you comply with.

**Does it work with my existing QGIS projects?**

Yes, the plugin works with any QGIS project. Simply load your data and use the exporter to produce compliant outputs.

**Can I customize the templates?**

Yes, you can create custom templates or modify existing ones to match your organization's requirements.

**What happens when standards change?**

We update the plugin to reflect changes in standards and regulations. Licensed users receive updates as part of their subscription.

**Is training included?**

Basic documentation is included with all licenses. Formal training is available as an additional service.
